# Character portraits

Drop one PNG per character here, named by character id (see portraits.js CHARACTERS):
mario.png, pam.png, lonnie.png, helen.png, frances.png, ronald-pumps.png

Any square image works; they're framed as head/torso "standees" in the card UI. A missing
file just hides the slot (no broken icon). Current art is 1080x1080 solid-white-bg renders.
